# Andrago Turbo

Статический сайт персонального тренера Андрея Батурина для GitHub Pages.

## Публикация

1. Загрузите все файлы из архива в корень репозитория.
2. Откройте `Settings → Pages`.
3. Выберите `Deploy from a branch`, ветку `main`, папку `/ (root)`.
4. Сохраните настройки.

Файл `CNAME` намеренно не добавлен: основной домен подключается только после проверки тестовой версии.
