УНИВЕРСАЛЬНЫЕ FAVICON ДЛЯ ANDRAGOTURBO.RU

Что покрывает комплект:
- Google Search
- Яндекс Поиск
- Chrome, Edge, Firefox, Opera
- Safari на Mac
- Safari и ярлык на iPhone/iPad
- Android и установка сайта на главный экран
- Windows и старые браузеры

1. Загрузите все файлы из архива в корень репозитория рядом с index.html.

2. В index.html удалите старый блок favicon:
   от первой строки <link rel="icon"...>
   до строки <link rel="manifest"...>

3. Удалите старую строку:
   <meta name="theme-color" content="#ffffff">

4. Вставьте содержимое favicon-head-snippet.html внутрь <head>.

5. После публикации проверьте:
   https://andragoturbo.ru/favicon.svg
   https://andragoturbo.ru/favicon-120x120.png
   https://andragoturbo.ru/favicon.ico
   https://andragoturbo.ru/apple-touch-icon.png
   https://andragoturbo.ru/site.webmanifest

6. Google Search Console:
   Проверка URL → https://andragoturbo.ru/ → Запросить индексирование.

7. Яндекс Вебмастер:
   Индексирование → Переобход страниц → https://andragoturbo.ru/

Поисковые системы обновляют значок не мгновенно.
